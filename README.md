# VerySoBot
NotSoBot proxy for Keybase

# Usage
With docker:
```
docker run -e KEYBASE_USERNAME=verysobot -e KEYBASE_PAPERKEY="keybase paperkey" -e DISCORD_TOKEN="discord token" fogapod/verysobot
```
